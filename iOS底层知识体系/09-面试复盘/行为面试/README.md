## 面试前建议顺序（约 2～3 小时一轮）

1. **先定 3 条「主素材」**：最好项目 ×1、最大挑战 ×1、最难技术题 ×1；**三条尽量来自不同项目**，避免同一个故事回答所有题显得单薄。
2. **每条写满 STAR**：各用一页纸（或备忘录），数字必须真实可核对（崩溃率、耗时、DAU、版本周期等）。
3. **每条准备 5 个追问**：若不是你一人完成，怎么分工？若重来会怎么做？若资源减半呢？
4. **录音限时 2 分钟自述**各一遍；超时则删形容词、加数字与决策点。
