## 使用方式

1. 按 P0 → P1 → P2 的顺序逐个攻克
2. 每篇笔记的 HTML 注释中已包含「建议涵盖的知识点」，作为写作提纲
3. 写完一篇后，回到此文件更新状态标记
4. 面试后及时更新「09-面试复盘」中的高频题和面经
5. 旧笔记在项目根目录的 `读书笔记/` 中，可作为参考素材
